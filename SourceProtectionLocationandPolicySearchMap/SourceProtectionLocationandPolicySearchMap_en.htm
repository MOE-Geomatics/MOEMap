Source Protection Information and Policy Search

This interactive map allows the user to search a location and view whether it is within a Source Protection Vulnerable Area. Source Protection Policies (password protected) to be used in addressing threats within those areas will display (where applicable) within a popup.

Additional Resources

To identify which threat is applicable for each vulnerable area, access the table of threat activities. Assessment Reports or Plans are also available for review.

To determine which measure may be used to address which threats, see the risk measures catalogue.
